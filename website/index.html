Everything you'd expect, nothing you don't

🕑

Automatic history

Every text clip you copy is captured automatically and kept newest-first.

🔎

Search & keyboard-first

Type to filter, ↑/↓ to move, ↵ to paste, ⌘1–9 for a quick pick.

⌨️

Global hotkey

Open the popup from any app with a shortcut you can rebind in Settings.

⚡

Paste directly

Pick a clip and it pastes straight into the app you were using.

📌

Pin favorites

Pin the clips you reuse so they stick to the top and never expire.

💾

Remembers across restarts

History is saved locally and restored the next time you launch.

🔒

Privacy-respecting

Skips passwords and secrets flagged by password managers. Nothing leaves your Mac.

🤖

Shortcuts integration

Automate clips with the macOS Shortcuts app via built-in App Intents.

Install with one command

Homebrew 
brew install --cask thkobierecki/tap/clipboard-manager

Signed & notarized, so it launches with no Gatekeeper warnings. Requires macOS 14+. To paste directly into apps, grant Accessibility access when prompted.

Or build from source

  1. Clone the repo

    git clone https://github.com/thkobierecki/mac-clipboard.git
cd mac-clipboard
open ClipboardManager.xcodeproj

  2. Build & run

    In Xcode, set a signing Team (your Apple ID) under Signing & Capabilities, then press ⌘R. A clipboard icon appears in your menu bar.

  3. Keep it around

    Drag the built ClipboardManager.app into /Applications and tick Launch at login in its menu.
